矩阵总结

AgenticHR 的核心优势

  • 零成本自托管 — MIT开源，数据完全自主
  • Boss直聘深度集成 — 自动采集+PDF下载，竞品无此能力
  • 腾讯会议自动化 — 自动创建会议，无需手动操作
  • 飞书深度集成 — 消息+日历+忙闲一体化
  • 轻量易部署 — SQLite+单进程，5分钟启动

AgenticHR 的主要差距

  • 无人才库 — 所有竞品的标配功能
  • 无数据报表 — 招聘漏斗、渠道分析、效率统计
  • 无Offer管理 — 缺少从面试到入职的闭环
  • 无面试评价 — 面试官无法提交结构化反馈
  • 单渠道限制 — 仅支持Boss直聘
  • 无AI面试 — 竞品的差异化重点

差距分析与建议路线图

基于竞品对比，以下是 AgenticHR 需要补充的功能，按优先级排序。P0 = 上线基本要求，P1 = 近期迭代，P2 = 中长期规划。

P0 - 必须

招聘数据看板与报表

竞品全部具备。至少需要：招聘漏斗（投递→筛选→面试→Offer→入职各阶段转化率）、岗位招聘进度、时间效率统计（平均筛选时长、面试周期等）。建议在 Dashboard 页面集成可视化图表。

P0 - 必须

面试评价/面评系统

面试官提交结构化面评（评分+维度评价+录用建议）是招聘闭环的关键环节。当前系统面试安排后无反馈收集机制。建议：每场面试支持面试官填写面评表，HR可查看汇总。

P0 - 必须

候选人状态流转完善

当前状态仅有 pending/passed/rejected/scheduled/cancelled。需要完善为完整招聘管道：新投递 → 筛选中 → 初筛通过 → 面试中 → 面试通过 → Offer阶段 → 已入职 / 已淘汰。每个状态变更需记录时间和操作人。

P1 - 近期

人才库管理

北森人才库利用率提升30%，Moka智能人才库是核心卖点。需要：已淘汰/已拒绝候选人不删除而是入库，支持标签分类、搜索、二次激活（重新进入招聘流程）。这是简历资产沉淀的关键。

P1 - 近期

Offer 管理流程

面试通过后的 Offer 发放、审批、候选人确认流程。竞品标配。建议：Offer模板管理、薪资审批流、候选人接受/拒绝跟踪。

P1 - 近期

操作日志与审计

多用户系统必须有操作审计。记录谁在什么时间做了什么操作（删除简历、修改面试、发送通知等）。用于安全合规和问题追溯。

P1 - 近期

简历/候选人详情页

竞品都有独立的候选人360度视图页面，集中展示：基本信息、简历原文、AI评估结果、面试记录、面评、通知记录、状态变更历史。当前系统信息分散在多个页面。

P2 - 中长期

多渠道简历采集

北森支持170+渠道。AgenticHR仅支持Boss直聘。可扩展支持：猎聘、智联、前程无忧、拉勾等主流平台的Chrome扩展采集，以及邮件简历自动导入。

P2 - 中长期

招聘官网/职位发布页

自动生成企业招聘页面，候选人可在线投递。竞品标配。可作为独立的前端页面，嵌入企业官网。

P2 - 中长期

AI面试能力

北森AI面试官是2026标杆，Moka提供16维度评估。可考虑接入大模型做AI初面：候选人通过链接进入AI面试，系统自动提问+录音+评估。技术门槛较高但差异化明显。

P2 - 中长期

移动端适配

面试官和HR常在手机端操作。当前系统仅桌面端。需要响应式设计或小程序/H5。

P2 - 中长期

入职管理模块

从Offer确认到员工报到的全流程：入职材料收集、入职手续清单、新人引导。竞品标配但实现周期较长。

Moka — 智能化招聘管理系统

公司概况

  • 成立: 2015年
  • 融资: C轮 1亿美元 (2021)
  • 投资方: GGV、金沙江、高瓴、老虎基金
  • 目标市场: 中大型企业
  • 定价: ~2万元/席/年，有最低席位要求
  • 部署: 仅SaaS

核心能力

  • Moka Eva: 基于大语言模型的AI HR产品，号称10倍提效
  • 简历解析: 自研技术，支持word/图片/pdf/excel，准确率90%+
  • AI JD生成: 18个行业模板，可自定义
  • AI面试: 量身定制面试题目，16维度能力评估
  • 全渠道聚合: 一站式发布+简历统一汇总
  • 智能初筛: 自动完成70%简历初筛
  • 协同面试: 面试日历双向同步
  • 招聘数据分析: 完整BI报表

Moka 赢在哪里

  • AI深度: Eva产品矩阵远超我们的AI能力
  • 全渠道: 我们仅Boss直聘
  • 企业成熟度: 大客户案例丰富，流程完善
  • 数据分析: 完整的招聘BI，我们完全没有

我们赢在哪里

  • 零成本: Moka 2万/席/年，我们免费开源
  • Boss深度集成: Moka无法自动采集Boss简历
  • 自托管: 数据完全自主，无厂商锁定
  • 轻量: 5分钟部署 vs 企业级实施周期
  • 可定制: 开源可二次开发
应对话术：当客户说"Moka功能更全"

"Moka确实功能全面，但每席位年费2万起步，10个HR就是20万/年。AgenticHR完全免费开源，部署在你自己的服务器上，数据不出企业。对于中小团队，我们覆盖了招聘核心流程（采集→筛选→面试→通知），而且Boss直聘的自动化采集是Moka做不到的。"

  • 定位: 中国HR SaaS龙头
  • 渠道: 对接170+外部招聘渠道
  • 上市: 港股上市
  • 目标市场: 中大型企业
  • 定价: 企业定制，价格较高
  • 部署: SaaS + 支持私有化

核心能力

  • 北森AI Family: 10大AI助手 + 500+ AI特性
  • AI面试官: 2026年校招/社招标杆应用
  • AI人才库: 智能分类、透视分析、精准推荐、定向激活，利用率提升30%
  • 170+渠道: 覆盖智联、前程、58、Boss、国聘等
  • 一体化: 招聘→测评→入职→绩效→学习全链路
  • 招测一体: 招聘+人才测评结合

北森赢在哪里

  • AI面试官: 行业标杆，我们完全没有
  • 人才测评: 心理测评+能力评估一体化
  • 渠道覆盖: 170+ vs 我们仅1个
  • 全链路: 从招聘到绩效全覆盖

我们赢在哪里

  • 零成本: 北森报价动辄数十万/年
  • 实施速度: 5分钟 vs 数月实施周期
  • Boss自动化: 北森需要手动操作Boss
  • 灵活定制: 开源可改 vs 定制需求排期
应对话术：当客户说"北森是行业龙头"

"北森是大厂首选没错，但实施周期3-6个月，年费用数十万。我们面向的是'现在就要开始招人'的中小团队——今天部署，明天就能从Boss直聘自动采集简历。等公司规模上去了再考虑切换到企业级方案也不迟，而且数据随时可以导出。"

飞书招聘 — 一体化协作招聘

  • 厂商: 字节跳动
  • 定位: 飞书生态内置招聘模块
  • 特色: 深度打通日历、音视频、群聊
  • 目标市场: 飞书用户企业
  • 定价: 飞书企业套件内含
  • 部署: 仅SaaS

核心能力

  • 三合一工作台: 简历+面试+面评同步操作
  • AI能力: 简历评估批量筛选、人岗匹配、AI面试题推荐
  • 实时语音转文字: 面试实时转写
  • AI面试总结: 面试后自动生成总结
  • 在线代码考核: 技术岗在线编程面试
  • 原生协作: 与飞书日历、会议、文档无缝打通

飞书招聘赢在哪里

  • 协作深度: 原生打通飞书全套工具
  • 面试体验: 三合一工作台+实时转写
  • AI面试总结: 自动总结面试内容
  • 多渠道: 支持多招聘平台

我们赢在哪里

  • 不锁定生态: 飞书招聘仅限飞书用户
  • Boss深度自动化: 飞书招聘无此能力
  • 腾讯会议支持: 不限于飞书视频会议
  • 自托管+开源: 数据自主
  • 成本: 免费 vs 飞书企业版费用
应对话术：当客户说"我们用飞书，直接用飞书招聘就好了"

"飞书招聘确实和飞书协作体验很好，但AgenticHR的优势是Boss直聘自动化——自动采集简历、自动求简历、PDF解析入库，这些飞书招聘都做不到。而且我们也深度集成了飞书通知和日历，可以作为飞书招聘的有力补充。很多HR是Boss直聘+飞书协作双线并用的。"

  • 厂商: Boss直聘 (看准科技)
  • 定位: 招聘平台增值服务
  • 2026春招: 日活峰值2350万
  • 目标市场: 所有企业
  • 定价: 按服务包购买
  • 部署: 仅平台内

核心能力

  • AI深度搜索: 面向高要求企业的AI精准搜索工具
  • AI面试总结: 30%招聘者使用AI辅助面试总结
  • AI托管筛选: 批量招聘场景AI自动筛选
  • Boss面试间: 内置视频面试
  • 协作招人: 多人协同管理
  • 筛选约面: 进度统计周报

Boss企业版赢在哪里

  • 流量优势: 日活2350万的候选人池

我们赢在哪里

  • 自动化采集: 我们能把Boss的数据提取出来
  • 跨平台管理: 统一管理多渠道候选人

"我们不是Boss直聘的替代品，而是它的增强工具。Boss直聘是你找人的地方，AgenticHR帮你把Boss上的简历自动化地管理起来——自动采集、AI评估、安排面试、发飞书通知、创腾讯会议，这些Boss直聘本身都不提供。"

开源赛道定位

全球开源ATS赛道中，没有任何一个项目同时具备：中国招聘平台集成（Boss直聘）、中国IM集成（飞书）、中国会议集成（腾讯会议）、AI简历评估。AgenticHR 是目前唯一面向中国中小企业的 AI 驱动开源招聘系统，这是一个明确的空白市场。

AgenticHR 独特价值

竞品做不到的事

  • Boss直聘自动化采集: Chrome扩展自动点击候选人、下载PDF简历、解析入库 — 任何竞品SaaS都不可能做到这一点（平台限制）
  • 完全自托管+MIT开源: 数据不出企业，无SaaS年费，可任意定制
  • 腾讯会议自动创建: 通过浏览器自动化创建会议并获取链接，竞品依赖手动操作或官方API（需企业认证）
  • 飞书深度通知: 消息+日历事件+忙闲查询三合一
  • 5分钟极速部署: SQLite免配置，单命令启动，vs 竞品数周到数月的实施周期

目标用户画像

  • 1-5人HR团队的中小企业
  • 主要从Boss直聘招人，不需要170个渠道
  • 使用飞书协作，需要面试通知到飞书
  • 预算有限，无法承担2万/席/年的SaaS费用
  • 注重数据隐私，希望候选人数据不存在第三方平台
  • 有基本技术能力（或有IT支持），能部署和维护

竞争定位一句话

"中小企业的免费AI招聘助手 — Boss直聘深度集成 + 飞书通知 + 自托管"
