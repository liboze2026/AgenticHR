# Milestone Execution Skill
1. Read the plan doc at docs/superpowers/plans/*.md for the current milestone
2. For each task: write failing test → implement → pass → commit
3. After all tasks: run `pnpm test && pnpm typecheck`
4. Check `git status` for untracked files and pnpm-lock.yaml changes before merge
5. Report summary with test count and commit count
